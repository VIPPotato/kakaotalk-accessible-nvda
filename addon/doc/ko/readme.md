# 카카오톡 접근성 향상 기능

## 이 애드온이 하는 일
카카오톡은 커스텀 목록 컨트롤을 사용하기 때문에, NVDA가 선택된 항목이 아니라 목록 컨테이너에 포커스를 유지하는 경우가 있습니다. 이 애드온은 연락처/채팅/메시지 목록 등에서 항목을 이동할 때 NVDA가 내용을 올바르게 읽도록 접근성을 개선합니다.

## 설치
1. GitHub 릴리스에서 최신 `.nvda-addon` 파일을 다운로드합니다.
2. 파일을 열고 NVDA에서 설치를 승인합니다.
3. 안내에 따라 NVDA를 재시작합니다.

## 사용 방법
추가 설정 없이 바로 동작합니다. 카카오톡 목록에서 방향키로 이동하면 NVDA가 현재 선택된 항목을 읽어야 합니다.

## 문제 해결
- 여전히 항목이 읽히지 않으면, 지원되는 NVDA 버전인지 확인하세요(“애드온 정보” 화면).
- 버그 제보 시 NVDA 버전, 카카오톡 버전, NVDA 로그(필요 시 “디버그” 로그)를 함께 첨부해 주세요.
